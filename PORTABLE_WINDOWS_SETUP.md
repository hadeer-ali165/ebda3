# تشغيل البرنامج على أي جهاز Windows بدون تثبيت سيرفر

هذا المشروع أصبح جاهزًا للتشغيل بنظام Portable.

## الملفات المهمة

- `StartApp.bat`: تشغيل البرنامج بضغطة واحدة.
- `StopApp.bat`: إيقاف البرنامج بعد الاستخدام.
- `database/database.sqlite`: قاعدة البيانات المحلية (SQLite).

## ما الذي يجب تسليمه مع المشروع؟

انسخي مجلد المشروع كاملًا للمستخدم، **ومعه** مجلد PHP Portable داخل:

- `runtime/php/php.exe`

> أي نسخة PHP Portable (Windows x64) تعمل، بشرط وجود `php.exe` في المسار السابق.

## أول تشغيل عند المستخدم

1. افتح `StartApp.bat`.
2. السكربت سيقوم تلقائيًا بـ:
   - تجهيز ملف قاعدة البيانات إذا غير موجود.
   - تشغيل الـ migrations.
   - تشغيل seeder للأدمن.
   - فتح البرنامج في المتصفح على: `http://127.0.0.1:8090`

## إغلاق البرنامج

- افتح `StopApp.bat`.

## بيانات الأدمن الافتراضية

- البريد: `admin@system.local`
- كلمة المرور: `Admin@123456`

## ملاحظات هامة للتوزيع

- لا يحتاج المستخدم تثبيت XAMPP أو MySQL أو Composer.
- يجب تسليم المجلد كاملًا كما هو (بما في ذلك `vendor` و`public/build` و`runtime/php`).
- إذا أردتِ نسخة "شكل برنامج نافذة مستقلة" بدون متصفح، يمكن لاحقًا تغليف نفس المشروع بـ PHP Desktop بسهولة.
